The Gradient

Blue gradient · the signature move

Every gradient across the site builds from the same 3-stop blue. Used on headline emphasis (ems), primary buttons, display numbers, section labels, and hover glows.

Abundance
Text Gradient · 3-stop

Deeper, richer blues. Applied to all headline emphasis words (em), stat numbers, and gradient labels.

linear-gradient(135deg,
  #1D4ED8 0%,
  #2563EB 50%,
  #3B82F6 100%)

Typography

Two fonts. One system.

Greycliff CF handles everything — body, headlines, labels. Neo Contact is used exclusively for emphasis accent words in headlines (inherited from the logo).

Aa
Greycliff CF
Primary brand typeface · via Adobe Fonts · all body, headlines, labels, UI
400 · Regular Body copy default weight
500 · Medium Labels, subtle emphasis
600 · SemiBold Secondary headlines
700 · Bold Primary headlines
800 · ExtraBold Display numbers, stats
Aa
Neo Contact LT Pro
Accent typeface · self-hosted · logo font · emphasis only
700 · Single weight AI-Powered Selling System
Usage Only used inside headline <em> elements with the blue gradient applied. Always 1.2× the surrounding Greycliff size. Always on its own line in headlines.

Wordmark lockups

Primary logos. Dark wordmark on light surfaces. Light wordmark on dark surfaces. Never place light-on-light or dark-on-dark.
